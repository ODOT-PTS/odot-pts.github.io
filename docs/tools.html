GTFS-ride Tools

The following tools can be used for creating, validating and analyzing GTFS-ride data.

  • node-gtfs-ride - A command-line utility for processing transit ridership data. It will import a GTFS file and Automated Passenger Counter (APC) data and then export ridership data in GTFS-ride format.
  • Ridership App - A tool for visualizing, analyzing and exporting transit ridership data in GTFS-ride format.
  • Feed Validator - Validates complete GTFS-ride feeds.
